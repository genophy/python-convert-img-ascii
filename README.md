# python-convert-img-ascii
### Description
- python convert the image to ascii characters.and the demo is base on other demo(not mine)
- see [beautiful-python-a-simple-ascii-art-generator-from-images](https://www.hackerearth.com/practice/notes/beautiful-python-a-simple-ascii-art-generator-from-images/)

### run it
```python
python3 pildemo.py demo.png > pildemo.txt
```
- this demo's file list:
 ```
  workdir/
      demo.png # not upload [python-logo](https://www.python.org/static/img/python-logo@2x.png)
      pildemo.py
```
